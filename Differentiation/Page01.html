Function is simply a rule that tells how something depends on another thing.

As an example you know that the earth rotates around the sun with time. Hence The position of the earth depends on time. So we say, the position of the earth is a function of time.

Here, the time \((t)\) is called the independant variable and, the position of the earth \((P)\) at time \((t)\) is the dependant variable.

Therefore, we denote

$$P\ =\ f(t)$$ which is said as " \(P\) is a function of \(t\)"

You may be not familiar with mathematical notations, but keep in your mind that this tells you that "The \(Position(P)\) is a function of \(Time(t)\)" or "At \(Time(t)\) the position of the earth is denoted by \(f(t)\)"

Some people think \(f(t)\) means \(f\) multiplied by \(t\) and that is a serious misunderstanding

Examples of some mathematical functions \begin{array}{l} y\ =\ f( x)\\ y=x^{2} \ +\ x\ +\ 25\\ f( x) \ =\ 4x^{3} +\ 56\\ f( t) \ =\ 3t^{2} \ +\ t\\ f( \theta ) \ =\ \sin( \theta ) \ +\ \cos( \theta ) \ +\ 4 \end{array}

Limit of a function \(f(x)\) defines the value of the function at a certain point at which the independent variable \((x)\) reaches some value \((a)\) (reaches = approximately equal)

We denote it by, $$\lim\limits _{n\rightarrow a} \ f( x) \ =\ L$$

which is said as "The limit of \(f(x)\) equals to \(L\) as \(x\) approaches \(a\)"

Let's illustrate this on \(xy\) plane
Let \(y\ =\ f(x)\) be any function of \(x\)

As you can see, when the value of \(x\) reaches (approximately equals to) \(a\), the value of the function reaches \(L\). Hence we say, "When \(x\) approaches \(a\), the limit of the function \(y\) equals to \(L\)"

We will consider a function \(y\ =\ f( x)\).
Recall that \(x\) is the independant variable and \(y\) is the dependant variable (because \(y\) depends on \(x\) such that it satisfies the conditions denoted by \(f)\))

• Firstly we will graph these data on an \(xy)\) plan.
(This plane is called the Cartesian Plane in mathematics)
\(x\) - axis shows the \(x\) values and \(y\) - axis shows the \(f( x)\) values

• Let's assume our graph looks like this when we graph \(x\) against \(f(x)\).

• Then we will select a point on the graph to find the gradient of the graph at this point.
(Gradients is also another term that simply tells the ratio of \(y\) to \(x\). that is \(y/x\))

• Then we take another point as depicted in this illustration and draw a straight line through the two points
• Next, we find the \(slope(=gradient))\) of the straight line as follows. (\(slope = \delta y/\delta x , \delta \) is used to denote a very small amount)

$$slope\ =\frac{\ difference\ between\ y\ values}{\ difference\ between\ x\ values}$$

$$slope\ =\frac{f( a\ +\ \delta a) \ -\ f( a)}{\ ( a\ +\ \delta a) \ -\ a}$$

$$slope\ =\frac{f( a\ +\ \delta a) \ -\ f( a)}{\delta a}$$

Let's replace \(a\) with \(x\), because \(a\) can be any value in the domain \(x\)

$$slope\ =\frac{\delta y}{\delta x} \ =\frac{f( x\ +\ \delta x) \ -\ f( x)}{\delta x}$$

Then we find the limit of the function at both sides when \(\delta x\) approaches \(0\)

$$\lim\limits _{\delta x\rightarrow 0} \frac{\delta y}{\delta x} \ =\lim\limits _{\delta x\rightarrow 0}\frac{f( x\ +\ \delta x) \ -\ f( x)}{\delta x}$$

Lets assume this equals to \(L\),

$$\lim\limits _{\delta x\rightarrow 0}\frac{\delta y}{\delta x} \ =L$$

Here, \(\lim\limits _{\delta x\rightarrow 0}\frac{\delta y}{\delta x} \) is called the derivative of the function \(f(x)\) at point \(x\) and is denoted by $$\frac{dy}{dx}$$

Finding the derivative of a function is called the Differentiation.

• Slope at point \(A = dy/ dx\), which means the derivative of the function at point \(A\) is \(dy/ dx\).

• Therefore you can see that Differentiation is practically used to find the rate of change of something happans. Such as flow of electricity, rates of change of population growth and growth of deceases etc.
