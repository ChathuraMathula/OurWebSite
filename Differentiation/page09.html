Product Rule \((U.V)\)

❯❯

Let's assume \(U\) and \(V\) be two funtions of \(x\) So, $$U\ =\ f_{1}( x) \ \ ,\ V\ =\ f_{2}( x)$$ Then the product of two funtions of \(x\) can be written as, $$U.V$$ If this is another funtion called \(y\) We can say, $$y = U.V$$
• Here, dot is used to denote the product of two functions
Then, if we want to find the derivative of this function w.r.t \(x\)
• w.r.t = with respect to
$$\frac{dy}{dx} \ =\frac{d( U.V)}{dx}$$ Then the Product Rule says, $$\frac{d( U.V)}{dx} \ =U.\frac{d( V)}{dx} \ +V.\frac{d( U)}{dx}$$
• Please refer the examples mentioned below to have a better understanding about this rule.

*****

Examples


Example 1
$$y\ =\ ( 5x\ +\ sin\ x) \ .\left( 2x^{2\ } +3\right)$$ Here we can take, $$U\ =\ 5x\ +\ sin\ x$$ and $$V\ =\ 2x^{2\ } +3$$ Then this equation is in the form of \(y =U.V\), By applying the product rule we have, $$\frac{dy}{dx} \ =U.\frac{d( V)}{dx} \ +V.\frac{d( U)}{dx}$$
$$\frac{dy}{dx} \ =( 5x\ +\ sin\ x) .\left\{\frac{d\left( 2x^{2\ } +3\right)}{dx}\right\}$$ $$+\left( 2x^{2\ } +3\right) .\left\{\frac{d( 5x\ +\ sin\ x)}{dx}\right\}$$
$$\frac{dy}{dx} \ =( 5x\ +\ sin\ x) .\left\{\frac{d\left( 2x^{2\ }\right)}{dx} +\frac{d( 3)}{dx}\right\}$$ $$+\left( 2x^{2\ } +3\right) .\left\{\frac{d( 5x)}{dx} +\frac{d( sin\ x)}{dx}\right\}$$

$$\frac{dy}{dx} \ =( 5x\ +\ sin\ x)( 4x+0)$$ $$+\left( 2x^{2\ } +3\right)( 5+cos\ x)$$
$$\frac{dy}{dx} \ =( 5x\ +\ sin\ x)( 4x) \ +\left( 2x^{2\ } +3\right)( 5+cos\ x)$$
$$\frac{dy}{dx} \ =\ ( 30\ +\ 2cos\ x) \ x^{2\ } \ +\ ( 4sin\ x) x$$ $$+\ ( 15\ +\ 3cos\ x)$$


Example 2
$$y\ =\ 2x^{2} .e^{2x}$$ $$U\ =\ \ 2x^{2} , V\ =\ \ e^{2x}$$ By applying the product rule we have, $$\frac{dy}{dx} \ =\frac{d\ \left( 2x^{2} .e^{2x}\right)}{dx}$$ $$\frac{dy}{dx} \ =2x^{2}\left\{\frac{d\ \left( e^{2x}\right)}{dx}\right\} +e^{2x}\left\{\frac{d\ \left( 2x^{2}\right)}{dx}\right\}$$ $$\frac{dy}{dx} \ =2x^{2} .2e^{2x} \ \ +\ e^{2x}( 4x)$$ $$\frac{dy}{dx} \ =4x^{2} e^{2x} \ \ +\ e^{2x}( 4x)$$ $$\frac{dy}{dx} \ =\left( 4x^{2} \ +\ 4x\right) e^{2x}$$ $$\frac{dy}{dx} \ =\left( x^{2} \ +\ x\right) 4e^{2x}$$
