The general form of an Exponential Function is $$y\ =\ a^{x}$$ where \(a\ \neq \ 1\) and \(a\ >\ 0\)
The Exponential Function \(y\ =\ e^{x}\) is called the Natural Exponential Function. Here \(e\) is a constant and it is called the Euler's constant. $$e\ \approx \ 2.71828...$$ Which means \(e\) is approximately equal to \(2.71828\)

Let's consider the exponential function, $$y\ =\ a^{x}$$ By taking logarithms in base \(e\), of both sides, we have, $$\log_{e} \ y\ =\ \log_{e} \ \left( a^{x}\right)$$
In logarithm, \(\log_{a} \ \left( x^{r}\right) \ =r\ \log_{a} \ x\)
So, by applying this rule, $$\log_{e} \ y\ =\ x\log_{e} \ a$$
In logarithm, \(\log_{10} \ A\ =\lg A\)
At the same times, \(\log_{e} \ A\ =\ln A\)
Hence we write, $$\log_{e} \ y\ \ =\ x\ \ln a$$
Again in logarithm, \(\log_{a} \ y\ =b\) can be written as \(y\ =\ a^{b}\)
So, we have, $$y\ =\ e^{\ x\ \ln a}$$

1. Derivative of \(e^{x}\)e to x

$$\frac{d\left( e^{x}\right)}{dx} \ =e^{x}$$

2. Derivative of \(e^{kx}\)e to x

$$\frac{d\left( e^{kx}\right)}{dx} \ =ke^{kx}$$

Now you get the idea that we cannot differentiate \(y = a^{x}\) directly.
To do so, we have to rewrite this equation as mentioned above.
Hence, now $$y\ =\ e^{\ x\ \ln a}$$ Then you have to differentiate this equation with respect to \(x\)
That is, $$\frac{dy}{dx} \ =\frac{d\left( e^{\ x\ \ln a}\right)}{dx}$$ • To differentiate this you must know the chain rule.

Example 1
$$y\ =2\ e^{\ x}$$ differentiate with respect to \(x\)x, we have $$\frac{dy}{dx} \ =\frac{d\left( 2\ e^{\ x}\right)}{dx}$$ $$\frac{dy}{dx} \ =2\frac{d\left( e^{\ x}\right)}{dx}$$ Hence, We have, $$\frac{dy}{dx} \ =2\ e^{\ x}$$


Example 2
$$y\ =\ 4\left( x\ +\ \ e^{\ x}\right) \ +\ cos\ x$$ differentiate with respect to \(x\)x, we have $$\frac{dy}{dx} \ =\frac{d\left\{4\left( x\ +\ \ e^{\ x}\right)\right\}}{dx} \ +\frac{d( cos\ x)}{dx}$$ $$\frac{dy}{dx} \ =\frac{d( 4x)}{dx} \ +\frac{d\left( 4e^{\ x}\right)}{dx} \ +\frac{d( cos\ x)}{dx}$$ $$\frac{dy}{dx} \ =4\ +\ 4x\ -\ sin\ x$$ Hence, We have, $$\frac{dy}{dx} \ =4( 1+x) \ -sin\ x$$


Example 3
$$y\ =\ x^{2} \ +\ 2e^{x} \ -\ cot^{-1} \ x$$ differentiate with respect to \(x\)x, we have $$\frac{dy}{dx} \ =\frac{d\left( x^{2}\right)}{dx} \ +\frac{d\left( 2e^{x}\right)}{dx} \ -\frac{d\left( cot^{-1} \ x\right)}{dx}$$ $$\frac{dy}{dx} \ =\ 2x\ +\ 2e^{x} \ -\left\{\frac{-1}{\sqrt{1-x^{2}}}\right\}$$ Hence, We have, $$\frac{dy}{dx} \ =\ 2\left( x\ +\ e^{x}\right) +\ \frac{1}{\sqrt{1-x^{2}}}$$
