අවකලනය \((dy/dx)\)
පටන් ගැන්ම ශ්‍රිතය හැඳින්වීම සීමාව හැඳින්වීම ව්‍යුත්පන්නය හැඳින්වීම මූලික අවකලන නීති ත්‍රිකෝණමිතික ශ්‍රිත ප්‍රතිලෝම ත්‍රිකෝණමිතික ශ්‍රිත ඝාතීය ශ්‍රිත ලඝු ශ්‍රිත ගුණ කිරීමේ නීතිය \((U.V)\) බෙදීමේ නීතිය \((U/V)\) දාම නීතිය අනුයාත අවකලනය අධ්‍යාහෘත අවකලනය පරාමිතික අවකලනය අභ්‍යාස
❮❮

ඝාතීය ශ්‍රිත

❯❯

  • ඝාතීය ශ්‍රිත වල සාධාරණ ආකාරය පහත ලෙස වේ.
$$y\ =\ a^{x}$$
(එනම් \(a\ \neq \ 1\) සහ \(a\ >\ 0\))
  • The Function \(y\ =\ e^{x}\) ශ්‍රිතය Natural Exponential Function ලෙස හඳුන්වනු ලබයි.

  • මෙහි, \(e\) යනු නියතයක් වන අතර එයට ඔයිලර් නියතය (Euler's constant) ලෙස කියනු ලබයි.
$$e\ \approx \ 2.71828...$$ මෙහි තේරුම \(e\) හි අගය \(2.71828\) ට ආසන්න වශයෙන් සමානයි යන්නයි.

දැන් අප ඝාතීය ශ්‍රිතයක් සලකමු, $$y\ =\ a^{x}$$ මෙහි දෙපසෙහිම \(e\) පාදයට ලඝු සැලකීමෙන් $$\log_{e} \ y\ =\ \log_{e} \ \left( a^{x}\right)$$
ලඝුගණක වලදී, \(\log_{a} \ \left( x^{r}\right) \ =r\ \log_{a} \ x\) ලෙස නීතියක් ඇත.

එම නීතිය භාවිතයෙන්, $$\log_{e} \ y\ =\ x\log_{e} \ a$$
ලඝුගණක වලදී, \(\log_{10} \ A\ =\lg A\)
එසේම, \(\log_{e} \ A\ =\ln A\)

එබැවින් පහත පරිදි ලිවිය හැක, $$\log_{e} \ y\ \ =\ x\ \ln a$$
නැවතත් ලඝුගණක නීති භාවිතයෙන්, \(\log_{a} \ y\ =b\) යන්න \(y\ =\ a^{b}\) පරිධි ලිවිය හැක.

එමනිසා, $$y\ =\ e^{\ x\ \ln a}$$

1. \(e^{x}\) හි ව්‍යුත්පන්නයe to x

$$\frac{d\left( e^{x}\right)}{dx} \ =e^{x}$$


2. \(e^{kx}\) හි ව්‍යුත්පන්නයe to x

$$\frac{d\left( e^{kx}\right)}{dx} \ =ke^{kx}$$

දැන් \(y = a^{x}\) යන්න සෘජුවම \(x\) විෂයෙන් අවකලනය කළ නොහැකි බව ඔබට තේරුම් යනු ඇත.
එසේනම් ඒ සඳහා අපගේ ප්‍රකාශනය ඉහත පරිධි නැවත සකසා ගත යුතුය.
එමනිසා, $$y\ =\ e^{\ x\ \ln a}$$
දැන් මෙය \(x\) විෂයෙන් අවකලනය කළ හැක, $$\frac{dy}{dx} \ =\frac{d\left( e^{\ x\ \ln a}\right)}{dx}$$ • මෙහි අවකලනය ලබා ගැනීමේදී දාම නීතිය පිළිබඳ දැනුම ඔබට තිබිය යුතුය. ඒ සඳහා ඉදිරි පරිච්ඡේද අධ්‍යනය කරන්න.

නිදසුන 1
$$y\ =2\ e^{\ x}$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\frac{d\left( 2\ e^{\ x}\right)}{dx}$$ $$\frac{dy}{dx} \ =2\frac{d\left( e^{\ x}\right)}{dx}$$ එමනිසා, $$\frac{dy}{dx} \ =2\ e^{\ x}$$


නිදසුන 2
$$y\ =\ 4\left( x\ +\ \ e^{\ x}\right) \ +\ cos\ x$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\frac{d\left\{4\left( x\ +\ \ e^{\ x}\right)\right\}}{dx} \ +\frac{d( cos\ x)}{dx}$$ $$\frac{dy}{dx} \ =\frac{d( 4x)}{dx} \ +\frac{d\left( 4e^{\ x}\right)}{dx} \ +\frac{d( cos\ x)}{dx}$$ $$\frac{dy}{dx} \ =4\ +\ 4x\ -\ sin\ x$$ එමනිසා, $$\frac{dy}{dx} \ =4( 1+x) \ -sin\ x$$


නිදසුන 3
$$y\ =\ x^{2} \ +\ 2e^{x} \ -\ cot^{-1} \ x$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\frac{d\left( x^{2}\right)}{dx} \ +\frac{d\left( 2e^{x}\right)}{dx} \ -\frac{d\left( cot^{-1} \ x\right)}{dx}$$ $$\frac{dy}{dx} \ =\ 2x\ +\ 2e^{x} \ -\left\{\frac{-1}{\sqrt{1-x^{2}}}\right\}$$ එමනිසා, $$\frac{dy}{dx} \ =\ 2\left( x\ +\ e^{x}\right) +\ \frac{1}{\sqrt{1-x^{2}}}$$
