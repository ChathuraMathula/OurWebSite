අවකලනය \((dy/dx)\)
පටන් ගැන්ම ශ්‍රිතය හැඳින්වීම සීමාව හැඳින්වීම ව්‍යුත්පන්නය හැඳින්වීම මූලික අවකලන නීති ත්‍රිකෝණමිතික ශ්‍රිත ප්‍රතිලෝම ත්‍රිකෝණමිතික ශ්‍රිත ඝාතීය ශ්‍රිත ලඝු ශ්‍රිත ගුණ කිරීමේ නීතිය \((U.V)\) බෙදීමේ නීතිය \((U/V)\) දාම නීතිය අනුයාත අවකලනය අධ්‍යාහෘත අවකලනය පරාමිතික අවකලනය අභ්‍යාස

ත්‍රිකෝණමිතික ශ්‍රිත

❯❯

  • සෘජුකෝණී ත්‍රිකෝණයක පාද සහ කෝණ අතර සම්බන්ධයන් අනුසාරයෙන් ත්‍රිකෝණමිතික න්‍යායන් ඉදිරිපත් කර ඇත.

  • ත්‍රිකෝණමිතික අනුපාත අනුබද්ධයෙන් ත්‍රිකෝණමිතික ශ්‍රිත ගොඩනගනු ලබයි.
පහත රූපයෙන් දැක්වෙන \(ABC\) සෘජුකෝණී ත්‍රිකෝණය සලකමු.

  • සෘජුකෝණී ත්‍රිකෝණයක සෘජුකෝණයට සම්මුකව (ඉදිරියෙන්) පිහිටන පාදයට කර්ණය (Hypotenuse) යැයි කියනු ලබයි.
$$sin\ \hat{A} \ =\ \frac{Opposite\ Side}{Hypotenuse} \ =\ \frac{BC}{AB}$$ $$cos\ \hat{A} \ =\ \frac{Adjacent\ Side}{Hypotenuse} \ =\ \frac{AC}{AB}$$ $$tan\ \hat{A} \ =\ \frac{Opposite\ Side}{Adjacent\ Side} \ =\ \frac{BC}{AC}$$
$$cosec\ \hat{A} \ \ =\ \frac{1}{sin\ \hat{A} \ }$$ $$sec\ \hat{A} \ \ =\ \frac{1}{cos\ \hat{A} \ }$$ $$cot\ \hat{A} \ \ =\ \frac{1}{tan\ \hat{A} \ }$$

1. \(sin (x)\) හි ව්‍යුත්පන්නයsin x

$$\frac{d( sin\ x)}{dx} \ =\ cos\ x$$

2. \(cos (x)\) හි ව්‍යුත්පන්නයcos x

$$\frac{d( cos\ x)}{dx} \ =\ -sin\ x$$

3. \(Tan (x)\) හි ව්‍යුත්පන්නයtan x

$$\frac{d( tan\ x)}{dx} \ =\ sec^{2} \ x$$

4. \(cosec (x)\) හි ව්‍යුත්පන්නයcosec x

$$\frac{d( cosec\ x)}{dx} \ =\ -cosec\ x\ .\ cot\ x$$

5. \(sec (x)\) හි ව්‍යුත්පන්නයsec x

$$\frac{d( sec\ x)}{dx} \ =\ sec\ x\ .\ tan\ x$$

6. \(cot (x)\) හි ව්‍යුත්පන්නයcot x

$$\frac{d( cot\ x)}{dx} \ =\ -cosec^{2} \ x$$

නිදසුන 1
$$f( x) \ =\ sin\ x\ +\ 3$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{d( f( x))}{dx} \ =\ \frac{d( sin\ x\ +\ 3)}{dx}$$ එකතුවක අවකලනයෙන්, $$\frac{d( f( x))}{dx} \ =\ \frac{d( sin\ x)}{dx} +\ \frac{d( 3)}{dx}$$ $$f'( x) \ =\ \frac{d( sin\ x)}{dx} +\ \frac{d( 3)}{dx}$$ නියතයක අවකලනයෙන්, $$f'( x) \ =\ \frac{d( sin\ x)}{dx} +\ 0$$ $$f'( x) \ =\ \frac{d( sin\ x)}{dx}$$ \(sin x\) හි ප්‍රථම ව්‍යුත්පන්නයෙන්, $$f'( x) \ =\ cos\ x$$


නිදසුන 2
$$y\ =\ cos\ x\ -\ 3x^{4} \ -\ 15$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ \frac{d\left( cos\ x\ -\ 3x^{4} \ -\ 15\right)}{dx}$$ වෙනසක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ \frac{d\left( 3x^{4}\right)}{dx} -\ \frac{d( 15)}{dx}$$ නියතයක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ \frac{d\left( 3x^{4}\right)}{dx} -\ 0$$ $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ \frac{d\left( 3x^{4}\right)}{dx}$$ නියතයකින් ගුණ කිරීමේදී අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ 3\times \left[\frac{d\left( x^{4}\right)}{dx}\right]$$ බලයක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ 3\times \left[ 4x^{( 4-1)}\right]$$ $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ 3\times 4x^{( 3)}$$ $$\frac{dy}{dx} \ =\ \frac{d( cos\ x)}{dx} -\ 12x^{3}$$ \(cos x\) හි ප්‍රථම ව්‍යුත්පන්නයෙන්, $$\frac{dy}{dx} \ =\ -sinx-\ 12x^{3}$$ එමනිසා, $$\frac{dy}{dx} \ =\ -\left( sinx\ +\ 12x^{3}\right)$$


නිදසුන 3
$$y\ =\ 2\left(\frac{sin\ x}{cos\ x}\right) \ +\ 31$$ මක්නිසාදයත්, $$\frac{sin\ x}{cos\ x} \ =\ tan\ x$$ $$y\ =\ 2( tan\ x) \ +\ 31$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ \frac{d( 2[ tan\ x] \ +\ 31)}{dx}$$ එකතුවක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( 2[ tan\ x])}{dx} +\ \frac{d( 31)}{dx}$$ නියතයක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( 2[ tan\ x])}{dx} +\ 0$$ නියතයකින් ගුණ කිරීමේදී අවකලනයෙන්, $$\frac{dy}{dx} \ =\ 2\ \times \left[\frac{d( tan\ x)}{dx}\right]$$ \(tan x\) හි ප්‍රථම ව්‍යුත්පන්නයෙන්, $$\frac{dy}{dx} \ =\ 2\ \times \left[ sec^{2} \ x\right]$$ $$\frac{dy}{dx} \ =\ 2\ sec^{2} \ x$$
