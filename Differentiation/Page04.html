Inverse Trigonometric Functions are used to obtain an angle from any of the angle's trigonometric ratio.


$$sin\ \hat{A} \ =\ \frac{Opposite\ Side}{Hypotenuse}$$ $$sin\ \hat{A} \ =\ \frac{BC}{AB}$$ $$\hat{A} \ =\ sin^{-1}\left(\frac{BC}{AB}\right)$$
\(\hat{A}\) is an angle.

*The symbol \(sin^{-1} X\) where \(X\) is a ratio,
should not be confused with \(( sin\ X)^{-1}\).

Keep in your mind that,
$$sin^{-1} X\ \neq \ ( sin\ X)^{-1}$$ $$( sin\ X)^{-1} \ =\ \frac{1}{sin\ X}$$
Let's see Domains of Inverse Functions.
Invers Function Domain
01. \(\theta \ =\ sin^{-1} X\)

02. \(\theta \ =\ cos^{-1} X\)
03. \(\theta \ =\ cosec^{-1} X\)

04. \(\theta \ =\ sec^{-1} X\)
05. \(\theta \ =\ tan^{-1} X\)

06. \(\theta \ =\ cot^{-1} X\)

1. Derivative of \(sin^{-1} X\)sin x

$$\frac{d\left( sin^{-1} X\right)}{dx} \ =\ \frac{1}{\sqrt{1-x^{2}}}$$

2. Derivative of \(cos^{-1} X\)cos x

$$\frac{d\left( cos^{-1} X\right)}{dx} \ =\ \frac{-1}{\sqrt{1-x^{2}}}$$

3. Derivative of \(tan^{-1} X\)tan x

$$\frac{d\left( tan^{-1} X\right)}{dx} \ =\ \frac{1}{1+x^{2}}$$

4. Derivative of \(cosec^{-1} X\)cosec x

$$\frac{d\left( cosec^{-1} X\right)}{dx} \ =\ \frac{-1}{ | x | \sqrt{x^{2} -\ 1}}$$

5. Derivative of \(sec^{-1} X\)sec x

$$\frac{d\left( sec^{-1} X\right)}{dx} \ =\ \frac{1}{ | x | \sqrt{x^{2} -\ 1}}$$

6. Derivative of \(cot^{-1} X\)cot x

$$\frac{d\left( cot^{-1} X\right)}{dx} \ =\ \frac{-1}{1+x^{2}}$$
\(| x |\) is the absolute value which is the non-negative value of \(x\).

For example,
$$| 3 | = 3$$ $$| -3 | = 3$$

Example 1
$$y\ =\ 5\ cos^{-1} x\ +\ 7\ sin^{-1} x$$ differentiate with respect to \(x\)x, we have $$\frac{dy}{dx} \ =\ 5\ \left[\frac{d\left( cos^{-1} x\right)}{dx}\right] \ +\ 7\ \left[\frac{d\left( sin^{-1} x\right)}{dx}\right]$$ $$\frac{dy}{dx} \ =\ 5\ \left[\frac{-1}{\sqrt{1-x^{2}}}\right] \ +\ 7\ \left[\frac{1}{\sqrt{1-x^{2}}}\right]$$ $$\frac{dy}{dx} \ =\ -5\ \times \ \left[\frac{1}{\sqrt{1-x^{2}}}\right] \ +\ 7\ \times \ \left[\frac{1}{\sqrt{1-x^{2}}}\right]$$ $$\frac{dy}{dx} \ =\ ( 7-5) \ \times \ \left[\frac{1}{\sqrt{1-x^{2}}}\right]$$ $$\frac{dy}{dx} \ =\ ( 2) \ \times \ \left[\frac{1}{\sqrt{1-x^{2}}}\right]$$ Hence, We have, $$\frac{dy}{dx} \ =\ \frac{2}{\sqrt{1-x^{2}}}$$


Example 2
$$y\ =cot^{-1} x\ +\ 8x\ +\ 2$$ differentiate with respect to \(x\)x, we have $$\frac{dy}{dx} \ =\frac{d\left( cot^{-1} x\ +\ 8x\ +\ 2\right)}{dx}$$ $$\frac{dy}{dx} \ =\frac{d\left( cot^{-1} x\right)}{dx} +\frac{d( 8x)}{dx} +\frac{d( 2)}{dx}$$ $$\frac{dy}{dx} \ =\left[\frac{-1}{1+x^{2}}\right] \ +\ ( 8\times 1) \ +\ 0$$ $$\frac{dy}{dx} \ =\left[\frac{-1}{1+x^{2}}\right] \ +\ 8$$ $$\frac{dy}{dx} \ =\frac{-1}{1+x^{2}} \ +\ \frac{8\ \times \left( 1+x^{2}\right)}{1+x^{2}}$$ $$\frac{dy}{dx} \ =\frac{-1\ +\ 8\ +\ 8x^{2}}{1+x^{2}}$$ Hence, We have, $$\frac{dy}{dx} \ =\frac{8x^{2} \ +\ 7}{1+x^{2}}$$


Example 3
$$y\ =cos^{-1} x\ +\ 5x^{2}$$ differentiate with respect to \(x\)x, we have $$\frac{dy}{dx} \ =\frac{d\left( cos^{-1} x\ +\ 5x^{2}\right)}{dx}$$ $$\frac{dy}{dx} \ =\frac{d\left( cos^{-1} x\right)}{dx} +\frac{d\left( 5x^{2}\right)}{dx}$$ $$\frac{dy}{dx} \ =\frac{-1}{\sqrt{1-x^{2}}} \ +\ 5\ \times \ \frac{d\left( x^{2}\right)}{dx}$$ $$\frac{dy}{dx} \ =\frac{-1}{\sqrt{1-x^{2}}} \ +\ 5\ \times \ 2x^{1}$$ Hence, We have, $$\frac{dy}{dx} \ =\frac{-1}{\sqrt{1-x^{2}}} \ +\ 10x$$
