අවකලනය \((dy/dx)\)
පටන් ගැන්ම ශ්‍රිතය හැඳින්වීම සීමාව හැඳින්වීම ව්‍යුත්පන්නය හැඳින්වීම මූලික අවකලන නීති ත්‍රිකෝණමිතික ශ්‍රිත ප්‍රතිලෝම ත්‍රිකෝණමිතික ශ්‍රිත ඝාතීය ශ්‍රිත ලඝු ශ්‍රිත ගුණ කිරීමේ නීතිය \((U.V)\) බෙදීමේ නීතිය \((U/V)\) දාම නීතිය අනුයාත අවකලනය අධ්‍යාහෘත අවකලනය පරාමිතික අවකලනය අභ්‍යාස
❮❮

මූලික අවකලන නීති

❯❯

මෙම කොටසේදී ඔබ අවකලනයේ මූලික නීති ඉගෙනගනු ලබයි.


  • \(f(x)\) යනු අප අවකලනය කිරීමට යන ශ්‍රිතය යැයි සලකමු.

  • \(f(x)\) ශ්‍රිතය \(x\) විෂයෙන් අවකලනය කිරීමෙන් පසු ලැබෙන පිළිතුර \(f^{\prime}(x)\) ලෙස නිරූපණය කරයි. එයද තවත් \(x\) හි ශ්‍රිතයක් වේ.

ගණිතයේදී අප මෙය පහත අයුරින් ප්‍රකාශ කරයි,

\(f(x)\) ශ්‍රිතයේ අවකල සංගුණකය \(f^{\prime}(x)\) වේ.


එයම පහත ලෙසද ප්‍රකාශ කළ හැක,

\(f(x)\) ශ්‍රිතයේ \(x\) විෂයෙන් පළමු වියුත්පන්නය \(f^{\prime}(x)\) වේ.


  • ශ්‍රිතයක් අවකලනය කිරීමේදී පිළිපැදිය යුතු මූලික නීතීන් පහත දක්වා ඇත. ඒව හොඳින් අධ්‍යයනය කර හොඳින් මතක තබා ගත යුතුය.

අවකලනයේ මූලික නීති

  1. නියතයක අවකලනය
  2. නියතයකින් ගුණකල විට අවකලනය
  3. බලයක අවකලනය
  4. එකතුවක අවකලනය
  5. වෙනසක අවකලනය

1. නියතයක අවකලනය.

  • ඕනෑම නියතයක් (\(= k\) යැයි සිතමු) ඕනෑම විෂයකින් අවකලනය කළවිට පිළිතුර \(0\) වේ.
\(y = k\) (නියතය) යන ශ්‍රිතය සලකන්න,
\(f( x) \ =\ k\)

\(\frac{dy}{dx} \ =0\ \) හෝ

\(\frac{d( f( x))}{dx} =\ 0\) හෝ

\(\frac{d( k)}{dx} \ =\ 0\) හෝ

\(f'( x) \ =\ 0\)

2. නියතයකින් ගුණකල විට අවකලනය

  • කිසියම් \(f(x)\) ශ්‍රිතයක් \(k\) නම් නියතයකින් ගුණ කර ඇත්නම් එම ශ්‍රිතයේ අවකලනය, නියතය වරක් ශ්‍රිතයේ අවකලනයයි \(k\times f'(x)\).
අප සලකන \(y\) නම් ශ්‍රිතය \(k\) නම් නියතය වරක් \(f(x)\) නම් තවත් ශ්‍රිතයක් නම්, $$y\ =\ k.f( x)$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ \frac{d( k.f( x))}{dx}$$ $$\frac{d( k.f( x))}{dx} \ =\ k.\frac{d( f( x))}{dx}$$ $$k.\frac{d( f( x))}{dx} \ =\ kf'( x)$$ එමනිසා, $$\frac{d( kf( x))}{dx} \ =\ kf'( x)$$

3. බලයක අවකලනය

  • \(x^{n}\) ආකාරයේ ප්‍රකාශනයක් අවකලනය කරන අයුරු මෙහිදී දැක්වේ.

පහත ලෙස \(y\) නම් වූ ශ්‍රිතයක් සලකමු (\(n\) යනු ඕනෑම සංඛ්‍යාවක් වේ) $$y\ =\ x^{n}$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ nx^{n-1}$$ එමනිසා, \(x^{n}\) ආකාරයේ ඕනෑම ශ්‍රිතයක පළමු ව්‍යුත්පන්නය \(nx^{n-1}\) ආකාරයේ තවත් ශ්‍රිතයක් වේ.

4. එකතුවක අවකලනය

  • ශ්‍රිත දෙකක් හෝ කිහිපයක එකතුවක අවකලනය එක් එක් ශ්‍රිතයේ අවකලනවල එකතුවට සමාන වේ.
\(u\) සහ \(v\) යනු \(x\) හි එකිනෙකට වෙනස් ශ්‍රිත දෙකක් යැයි ගනිමු.

එමනිසා, $$u\ =\ f_{1}( x) \ \ ,\ v\ =\ f_{2}( x)$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{du}{dx} \ =\ f'_{1}( x) \ \ ,\ \ \ \frac{dv}{dx} \ =\ f'_{2}( x)$$ අප සලකන ශ්‍රිතය තවත් ශ්‍රිත දෙකක එකතුවකින් සෑදී ඇත්නම් $$y\ =\ u\ +\ v$$ එබැවින්, $$\frac{dy}{dx} =\frac{d( u+v)}{dx}$$ $$\frac{d( u+v)}{dx} \ =\ \frac{du}{dx} \ +\ \frac{dv}{dx}$$

5.වෙනසක අවකලනය

  • ශ්‍රිත දෙකක් හෝ කිහිපයක වෙනසක අවකලනය එක් එක් ශ්‍රිතයේ අවකලනවල වෙනසට සමාන වේ.
\(u\) සහ \(v\) යනු \(x\) හි එකිනෙකට වෙනස් ශ්‍රිත දෙකක් යැයි ගනිමු.

එමනිසා, $$u\ =\ f_{1}( x) \ \ ,\ v\ =\ f_{2}( x)$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{du}{dx} \ =\ f'_{1}( x) \ \ ,\ \ \ \frac{dv}{dx} \ =\ f'_{2}( x)$$ අප සලකන ශ්‍රිතය තවත් ශ්‍රිත දෙකක වෙනසකින් සෑදී ඇත්නම් $$y\ =\ u\ -\ v$$ එබැවින්, $$\frac{dy}{dx} =\frac{d( u-v)}{dx}$$ $$\frac{d( u-v)}{dx} \ =\ \frac{du}{dx} \ -\ \frac{dv}{dx}$$

නිදසුන 1
$$y\ =\ 7x^{2} \ +\ 5$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ \frac{d\left( \ 7x^{2} \ +\ 5\right)}{dx}$$ එකතුවක අවකලනයේ නීතියෙන්, $$\frac{dy}{dx} \ =\ \frac{d\left( \ 7x^{2}\right)}{dx} \ +\ \frac{d( 5)}{dx}$$ නියතයකින් ගුණකල විට අවකලනයේ නීතියෙන්, $$\frac{dy}{dx} \ =\ 7\ \times \left[\frac{d\left( x^{2}\right)}{dx}\right] \ +\ \frac{d( 5)}{dx}$$ බලයක අවකලනයේ නීතියෙන්, $$\frac{dy}{dx} \ =\ 7\ \times \left[ 2x^{2-1}\right] \ +\ \frac{d( 5)}{dx}$$ $$\frac{dy}{dx} \ =\ 7\ \times 2x\ +\ \frac{d( 5)}{dx}$$ නියතයක අවකලනයේ නීතියෙන් $$\frac{dy}{dx} \ =\ 7\ \times 2x\ +\ 0$$ එමනිසා, $$\frac{dy}{dx} \ =\ 14x$$
නිදසුන 2
$$y\ =\ 10x^{3}$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ \frac{d\left( 10x^{3}\right)}{dx}$$ නියතයකින් ගුණකල විට අවකලනයේ නීතියෙන්, $$\frac{dy}{dx} \ =\ 10\ \times \ \left[\frac{d\left( x^{3}\right)}{dx}\right]$$ බලයක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ 10\ \times \ \left[ 3x^{( 3-1)}\right]$$ $$\frac{dy}{dx} \ =\ 10\ \times \ \left[ 3x^{2}\right]$$ එමනිසා, $$\frac{dy}{dx} \ =\ 30x^{2}$$
නිදසුන 3
$$y\ =\ 5x\ -\ 1$$ \(x\) විෂයෙන් අවකලනය කළවිට $$\frac{dy}{dx} \ =\ \frac{d( 5x\ -\ 1)}{dx}$$ වෙනසක අවකලනයෙන්, $$\frac{dy}{dx} \ =\ \frac{d( 5x)}{dx} -\frac{d( 1)}{dx}$$ නියතයකින් ගුණකල විට අවකලනයේ නීතියෙන්, $$\frac{dy}{dx} \ =5\ \times \ \left[\frac{d( x)}{dx}\right] -\frac{d( 1)}{dx}$$ බලයක අවකලනයෙන්, $$\frac{dy}{dx} \ =5\ \times \ \left[ x^{1-1}\right] -\frac{d( 1)}{dx}$$ $$\frac{dy}{dx} \ =5\ \times \ \left[ x^{0}\right] -\frac{d( 1)}{dx}$$ $$\frac{dy}{dx} \ =5\ \times \ [ 1] -\frac{d( 1)}{dx}$$ ඕනෑම සංඛ්‍යාවක \(0\) වන බලය \(1\) ට සමාන වේ. එනම් \(1^{0} =2^{0} =50^{0} =99999^{0} = 1\) නියතයක අවකලනයෙන්, $$\frac{dy}{dx} \ =5\ \times \ [ 1] -0$$ එමනිසා, $$\frac{dy}{dx} \ =5$$
