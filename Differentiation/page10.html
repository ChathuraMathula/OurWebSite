Quotient Rule \((U/V)\)

❯❯

Let's assume \(U\) and \(V\) be two funtions of \(x\) So, $$U\ =\ f_{1}( x) \ \ ,\ V\ =\ f_{2}( x)$$ Then the quotient of two funtions of \(x\) can be written as, $$\frac{U}{V}$$ If this is another funtion called \(y\) We can say, $$y\ =\frac{U}{V}$$ Then, if we want to find the derivative of this function w.r.t \(x\)
• w.r.t = with respect to
$$\frac{dy}{dx} \ =\frac{d\left(\frac{U}{V}\right)}{dx}$$ Then the Quotient Rule says, $$\frac{d\left(\frac{U}{V}\right)}{dx} \ =\frac{\ V.\frac{d( U)}{dx} \ -\ U\ \frac{d( V)}{dx}}{V^{2}}$$
• Please refer the examples mentioned below to have a better understanding about this rule.

*****

Examples


Example 1
$$y\ =\ \frac{5x^{2}}{3x}$$ Here we can take, $$U\ =\ 5x^{2} \ ,\ V\ =\ 3x$$ Then this equation is in the form of \(y\ =\frac{U}{V}\), By applying the quotient rule we have, $$\frac{dy}{dx} \ =\frac{d\left(\frac{5x^{2}}{3x}\right)}{dx}$$ $$\frac{dy}{dx} \ =\frac{\ V.\frac{d( U)}{dx} \ -\ U\ \frac{d( V)}{dx}}{V^{2}}$$ $$\frac{dy}{dx} \ =\frac{\ 3x.\frac{d\left( 5x^{2}\right)}{dx} \ -\ 5x^{2} \ \frac{d( 3x)}{dx}}{( 3x)^{2}}$$ $$\frac{dy}{dx} \ =\frac{\ 3x\ \times 10x\ -\ 5x^{2} \ \times 3}{( 3x)^{2}}$$ $$\frac{dy}{dx} \ =\frac{30x^{2} \ -\ 15x^{2}}{9x^{2}}$$ $$\frac{dy}{dx} \ =\frac{\ 15x^{2}}{9x^{2}}$$ $$\frac{dy}{dx} \ =\frac{15}{9}$$ $$\frac{dy}{dx} \ =\frac{5}{3}$$
