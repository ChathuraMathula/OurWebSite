Integration of the form \(x^{n}\)

  • Here you will see how to integrate the functions of the form of \(x^{n}\)

\(\int x^{n} \ =\ \frac{x^{( n+1)}}{n+1} \ +\ Constant\)

(where \(n ≠ -1\))

Checking the reverse Process

$$\frac{d\left(\frac{x^{( n+1)}}{n+1}\right)}{dx} \ =\ \frac{1}{( n+1)} \ \times \ ( n+\ 1) x^{n}$$ $$\frac{d\left(\frac{x^{( n+1)}}{n+1}\right)}{dx} \ =x^{n}$$

Practice Examples


Example 1 $$\int x^{3}dx \ =\ \frac{x^{( 3+1)}}{3+1} \ +\ Constant$$ $$\int x^{3}dx \ =\ \frac{x^{4}}{4} \ +\ Constant$$

Example 2 $$\int x^{3}dx \ =\ \frac{x^{\left(\frac{1}{3} \ +\ 1\right)}}{\frac{1}{3} +1} \ +\ Constant$$ $$\int x^{\left(\frac{1}{3}\right)}dx \ =\ \frac{3x^{\left(\frac{4}{3}\right)}}{4} \ +\ Constant$$

Example 3 $$\int \frac{1}{x^{5}}dx \ =\int x^{-5}dx$$ $$\int x^{-5}dx \ =\ \frac{x^{( -5+1)}}{-5+1} \ +\ Constant$$ $$\int x^{-5}dx \ =\ \frac{x^{( -4)}}{-4} \ +\ Constant$$ $$\int x^{-5}dx \ =\ -\frac{1}{4x^{4}} \ +\ Constant$$ $$\therefore \ \int \frac{1}{x^{5}}dx \ =-\frac{1}{4x^{4}} \ +\ Constant$$

("∴" is the mathematical symbol to denote the word "Therefore")


Example 4 $$\int _{1}^{2} x^{2} dx\ =\ \left\{\frac{x^{3}}{3}\right\}_{1}^{2}$$ $$\int _{1}^{2} x^{2} dx\ =\ \left\{\frac{2^{3}}{3}\right\} -\ \left\{\frac{1^{3}}{3}\right\}$$ $$\int _{1}^{2} x^{2} dx\ =\ \frac{8-1}{3}$$ $$\therefore \int _{1}^{2} x^{2} dx\ =\ \frac{7}{3}$$

Example 5 $$y\ =\ 2x^{3} -5x^{2} \ +\ x\ +\ 3$$ $$\int y\ dx\ =\ \int \left( 2x^{3} -5x^{2} \ +\ x\ +\ 3\right) \ dx$$
$$\int y\ dx=\int 2x^{3} dx-\int 5x^{2} dx$$ $$+\int x\ dx+\int 3dx\ +\ C$$
$$\int y\ dx=2\int x^{3} dx-5\int x^{2} dx$$ $$+\int x\ dx+3\int dx+\ C$$
$$\therefore \int y\ dx=\frac{2x^{4}}{4} -\frac{5x^{3}}{3} +\frac{x^{2}}{2} +3x+\ C$$

(\(C\ =\ Constant\))
